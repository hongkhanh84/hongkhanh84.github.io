Chào Mừng Đến Với Trang Web Của Tôi!

Đây là một trang web mẫu được host miễn phí trên GitHub Pages.

Bạn có thể chỉnh sửa file index.html này và thêm nội dung của riêng bạn (văn bản, hình ảnh, liên kết...).

Các bước để trang web này hiển thị trên tên miền GitHub của bạn:

  1. Lưu nội dung trên vào một file có tên index.html.
  2. Tạo một repository mới trên GitHub (hoặc sử dụng một repository đã có).
  3. Upload file index.html lên thư mục gốc (root) của repository, hoặc vào thư mục /docs (tùy thuộc cấu hình bạn muốn).
  4. Vào phần Cài đặt (Settings) của repository của bạn, tìm mục "Pages".
  5. Chọn branch mà bạn muốn GitHub Pages sử dụng (thường là main hoặc master). Chọn thư mục chứa code (/ (root) hoặc /docs). Lưu lại.
  6. GitHub sẽ bắt đầu build trang web của bạn. Sau vài phút, bạn có thể truy cập trang web tại địa chỉ tương ứng với cấu hình GitHub Pages của bạn (thường là tendangnhap.github.io hoặc tendangnhap.github.io/ten-cua-repository).
